# 🔍 Binary Search Questions in Java

> A curated collection of Binary Search problems solved in Java for coding interviews, placements, and competitive programming.

---

## 📌 About

This repository contains Java implementations of Binary Search algorithms and interview-focused problems. It covers fundamental concepts, advanced variations, and popular coding interview questions to help improve problem-solving skills.

---

## 🎯 Objectives

- Learn Binary Search fundamentals
- Master different Binary Search patterns
- Solve interview-oriented problems
- Improve algorithmic thinking
- Prepare for placements and coding contests

---

## ⏱️ Time Complexity

| Algorithm | Time Complexity |
|-----------|-----------------|
| Binary Search | O(log N) |
| Lower/Upper Bound | O(log N) |
| First & Last Occurrence | O(log N) |
| Search in Rotated Array | O(log N) |
| Search in Matrix | O(log(M × N)) |
| Binary Search on Answer | O(log Range × Check Function) |
